# Koinsavetask
Project Overview
This is a React Native Expo project with Firebase authentication, featuring login, signup, and dashboard screens. The app is responsive and works seamlessly on mobile, tablet, and desktop devices.

Features
 User authentication (login, signup) with Firebase
 Responsive design for mobile, tablet, and desktop
Dashboard screen with user information

Tech Stack
React Native Expo
Firebase Authentication

Setup Instructions

1. **Clone the repository**
```bash
git clone https://github.com/Tomex2023/Koinsavetask.git
cd Koinsavetask
2. Install dependencies: `npm install` or `yarn install`
3. Start the expo app: using `npx expo start ` at the terminal in the project.
